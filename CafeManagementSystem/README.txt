
Cafe Management System (Java Swing + SQLite)

1. Download sqlite-jdbc jar:
https://github.com/xerial/sqlite-jdbc/releases

2. Add jar to project library.

3. Open in NetBeans or IntelliJ.

4. Run CafeManagementGUI.java

Features:
- Add menu item
- Update menu item
- Delete menu item
- Search by name
- JTable
- Auto-create database (cafe.db)
- Validation
